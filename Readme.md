# Tabholic [1.0.0]
**Tabholic** is an information-focused mod for _ROUNDS_ that combines:
- The **convenience** of [Infoholic](https://thunderstore.io/c/rounds/p/Penial/Infoholic/)
- The **compatibility** of [TabInfo](https://thunderstore.io/c/rounds/p/willuwontu/TabInfo/)

It provides a highly customizable UI panel for displaying player stats and other useful information during gameplay.
## Features
- Displays player stats in a clean and customizable panel
- Fully customizable UI:
    - Panel position
    - Panel scale
    - Text spacing
    - Text color
    - And more
## How to Use
### Open Settings
1. Launch the game
2. Navigate to:  **Mods -> Tabholic**
### Customize the Info Panel
1. In the Tabholic settings menu, click:  **“Manage Tabholic Panel”**
2. This opens the **UI Manager Layout Manager**
3. Inside the Layout Manager:
    - Right-click the Tabholic panel
    - Select **“Open Options”** from the context menu
4. Adjust settings such as:
    - Text color
    - Spacing
    - Panel size
## Tips
- Changes are applied in real-time, so you can fine-tune your layout easily
- If the panel looks different in-game vs editor, this may be due to _ROUNDS_ post-processing
